типи даних
	- int
	- long
	- float
	- логічні
	- списки (масиви) a=['a','b','word']  - можна змінювати
	- кортеж (tuple) a=('a','b','word') - не можна змінювати 
	- множини (set) a={1,2} - невпордковані списки, містить тільки унікальні елементи - можна змінювати
		frozenset - не можна змінювати
	- словники (dict) - пари ключ:значення 
		{0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36}
	- None - нічого
